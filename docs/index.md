# 欢迎使用静态文档站点

## 简介

这是一个使用 [MkDocs](https://www.mkdocs.org/) 和 [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) 主题构建的静态文档站点。本站点提供了详细的使用指南、API参考和示例代码，帮助您快速入门和掌握相关功能。

## 主要功能

- 响应式设计，适配各种设备屏幕
- 强大的搜索功能，支持全文搜索
- 美观的文档布局和导航结构
- 代码高亮和语法显示
- 支持表格、标签页、提示框等丰富的文档元素

## 快速开始

1. 阅读 [安装指南](guide/installation.md) 了解如何设置本地开发环境
2. 查看 [快速入门](guide/quickstart.md) 学习基本使用方法
3. 探索 [API参考](api/overview.md) 获取详细的接口信息
4. 参考 [示例](examples/basic.md) 学习实际应用场景
