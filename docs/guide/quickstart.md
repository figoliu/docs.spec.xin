# 快速入门

本指南将帮助您快速了解如何使用和贡献到本静态文档站点。

## 浏览文档

文档站点的导航结构清晰，您可以通过左侧边栏浏览不同的章节：

- **首页**：提供站点概述和主要功能介绍
- **使用指南**：包含详细的操作说明和最佳实践
- **API参考**：提供所有API接口的详细文档
- **示例**：展示各种使用场景和代码示例
- **关于**：站点相关信息

## 搜索功能

您可以使用右上角的搜索框快速查找文档内容：

1. 点击搜索框或使用快捷键 `Ctrl + K`
2. 输入关键词
3. 从搜索结果中选择相关内容

搜索功能支持模糊匹配和高亮显示。

## 贡献文档

如果您想贡献文档，请按照以下步骤操作：

### 修改现有文档

1. 在本地编辑相应的 Markdown 文件
2. 运行 `mkdocs serve` 预览更改
3. 提交更改并创建 Pull Request

### 添加新文档

1. 在适当的目录下创建新的 Markdown 文件
2. 在 `mkdocs.yml` 中的 `nav` 部分添加新文档的链接
3. 运行 `mkdocs serve` 预览更改
4. 提交更改并创建 Pull Request

## 文档格式指南

在编写文档时，请遵循以下格式指南：

- 使用 Markdown 语法
- 标题层级清晰（#, ##, ### 等）
- 代码块使用 ``` 包围，并指定语言
- 使用表格展示结构化数据
- 使用提示框（admonition）突出重要信息
